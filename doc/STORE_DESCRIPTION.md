NextTab 是一款专注于提升浏览器效率的新标签页扩展。我们希望为你提供一个干净、现代且真正好用的起始页，让你在每次打开新标签时，都能更快地找到目标，专注于真正重要的事情。

为什么你应该安装 NextTab？
在日常浏览中，我们经常需要在繁杂的书签、历史记录和打开的标签页中寻找需要的信息。NextTab 将这些高频操作整合到了一个极简的界面中，通过键盘快捷键和智能的右键菜单，大幅减少你的鼠标点击次数。它完全在本地运行，不收集任何隐私数据。

核心功能：

⚡ 高效的快捷链接
• 自由添加常用网站，支持拖拽排序，打造你的专属导航。
• 智能右键菜单：右键点击快捷链接，即可快速查看该域名下的最近访问历史、相关书签和已打开的标签页，轻松找回之前的浏览进度。

🔍 强大的命令面板
• 随时按下 Cmd/Ctrl + K 唤起命令面板。
• 快速全局搜索你的历史记录、书签和标签页，无需在浏览器的各个菜单中翻找。

🎨 个性化与沉浸感
• 支持自定义壁纸，默认集成 Wallhaven 高质量壁纸库，让每次打开新标签页都是一种视觉享受。
• 极简的界面设计，没有多余的广告和信息流打扰。

🚀 便捷的扩展面板 (Popup)
• 无需打开新标签页，点击浏览器右上角的扩展图标，即可快速访问你的快捷链接。
• 一键将当前正在浏览的网页添加到快捷链接中。

⌨️ 键盘优先体验
• 完善的快捷键支持，让习惯使用键盘的你操作更加行云流水。

🔒 隐私至上
• 所有核心功能均在本地运行，你的浏览数据只属于你自己。

如果你厌倦了臃肿、充满广告的新标签页，或者想要一个能真正提升网页浏览效率的工具，NextTab 会是一个值得尝试的选择。
